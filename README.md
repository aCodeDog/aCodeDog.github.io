# Zifan Wang - Personal Academic Homepage

My personal academic homepage built with Jekyll, based on the [AcadHomepage](https://github.com/RayeRen/acad-homepage.github.io) template.

**Live site:** https://aCodeDog.github.io

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

Then visit http://localhost:4000

## Google Scholar Citations

Citation data is automatically updated via GitHub Actions. Set the `GOOGLE_SCHOLAR_ID` secret in your repository settings.
